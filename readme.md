## Ixibei PHP Spider

一、适用范围作用
    1，针对普通html列表，html详情采集
    2，json形式的数据返回列表，html详情采集
    3，混合形式的，列表形式采集（暂未开发）

二、目录结构
    1，src为项目代码
    2，tests为测试代码
    3，logs为日志代码
    
## install
{
    "require": {
        "ixibei/spider": "dev-master"
    }
}
